About this application

This application is a simple analysis tool for lingvistics, literary studies or literary theory. It helps in characterizing either lyrics or its author. The application is ment to handle poems, stories, and all sorts of textual material that is deemed literature, not excluding scientific, philosophical or historical material.

With statistical textual analysis one can understand better author's character, intentions, idiolect, mood, and many other factors that may help to construct a fuller view of the area of study. Or it can be used for sosiolingvistical study of social media messages to analyze ethnicity, religion, status, gender, level of education, age, etc. Words that we speak or write down tell one thing about us. But the explicitly written textual content doesn't always reveal everything - sometimes we may infer more from reading between the lines, or from statistical analysis.

For example, it has long been a mystery whether famous Old English epic poem Beowulf was composed by a single author or if it was stiched together from many textual sources [Guardian][The Harvard Gazette]. Recently an academic research team analyzed Beowulf statistically with a computer program and came to a conclusion that Beowulf was written by one author [Nature]. The program is written in Python and is freely and publicly available at GitHub.

For the moment this application only holds copyright free religious lyrics. The final purpose for this application is not set as yet, because this is a study lab project at the University of Helsinki . So, for instance, there is a join table Author which would be better suited if the authors formed a column in the Song table. Maybe in the future this application holds textual material that have many authors. Then it would be better to be as is. Moreover, a plan is to add a word frequency search by author, so the more important it is to have a table for them alone.

The point is: this is a school project, aimed at learning how to code. This is not a professional or commercial research tool. As to the religious hymns which are cultural loan material, they do have a lot common authors regardless of language. Therefore, a separate table for authors is justified.
